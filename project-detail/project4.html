Deteksi Penyakit pada Daun Pepaya Menggunakan Convolutional Neural Network dan Support Vector Machine

Okt 2024 - Des 2024

Data Analyst

In this project, I act as a Data Analyst responsible for the processing and analysis of papaya leaf image data to support the development of machine learning models in plant disease detection. This project aims to classify three main types of papaya leaf diseases—black spot, anthracnose, and curly leaves—automatically and accurately using an image processing-based approach.

My tasks include image data collection, data cleaning and annotation, and visual data exploration (EDA) to understand the characteristics of each disease class. I am also involved in image pre-processing such as normalization, image augmentation, and resolution standardization to ensure the data is ready for use in model training. The processed data is used as input for CNN and SVM-based models, with visual features extracted using the DenseNet architecture.

Through accurate and systematic data analysis, I help the team improve the quality of the dataset and the validity of the classification results, thereby supporting the development of a more efficient, accurate, and beneficial disease detection system for farmers.
